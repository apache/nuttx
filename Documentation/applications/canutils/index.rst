=============
CAN Utilities
=============

- lely-canopen - Lely CANopen support

- libdronecan - libcanard DroneCAN Library
- libobd2 - OBD-II Library
- libopencyphal - libcanard OpenCyphal Cyphal/CAN Library
- slcan - SocketCAN slcan tool
