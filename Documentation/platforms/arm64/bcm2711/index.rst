=======
BCM2711
=======

The `BCM2711 <https://www.raspberrypi.com/documentation/computers/processors.html#bcm2711>`_ is a Broadcom SoC used for
the Raspberry Pi 4B board.

- **CPU:** Quad-core ARM Cortex-A72
- **Interrupt Controller:** GIC400
