===============
TI TMS320-DM320
===============

(also called **DM320**)

NuttX operates on the ARM9 of
this dual core processor. This port uses the Neuros OSD
with a GNU arm-nuttx-elf toolchain\* under Linux or Cygwin. The port was
performed using the OSD v1.0, development board.
