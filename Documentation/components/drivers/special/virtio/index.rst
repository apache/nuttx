======
VirtIO
======

VirtIO (Virtual I/O) is a standardized interface for virtual devices in
paravirtualized environments. NuttX implements a complete VirtIO framework
based on OpenAMP, supporting various VirtIO drivers such as VirtIO-Net,
VirtIO-Block, VirtIO-Serial, and more.

The framework supports different transport layer implementations including
VirtIO-MMIO, VirtIO-PCI, and VirtIO-Remoteproc.
